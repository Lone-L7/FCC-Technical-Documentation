Intro

HTML is a markup language that has been used by millions of people around the globe.

HTML stands for Hyper Text Markup Language.

HTML elements tell the browser how to display the content.

Just like what you have seen right now, this website is created using HTML and CSS.

We will show you the basics of HTML.

HTML Tag
  • html
  • !DOCTYPE html
  • html lang="en"

The html tag itself defines root element of the website.

The !DOCTYPE html code defines that this document is HTML5 document.

While the html lang="en" code defines what language this document is.

Head Tag
  • head
  • meta charset="UTF-8"
  • meta http-equiv="X-UA-Compatible" content="IE=edge"
  • meta name="viewport" content="width=device-width, initial-scale=1.0"
  • title

The head tag contains the meta information about this page.

This tag, meta charset="UTF-8" attribute specifies the character encoding for the HTML document. The HTML5 specification encourages web developers to use the UTF-8 character set, which covers almost all of the characters and symbols in the world!
